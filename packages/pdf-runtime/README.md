# @intenttext/pdf-runtime (skeleton)

M1 runtime package skeleton for `renderHtml` and `createPdf` with explicit browser-pool controls.

Current status:

- typed API contracts
- core loader and validation/render wrappers
- browser pool control surface
- `createPdf` implementation scaffold with dynamic puppeteer import

This package is intentionally minimal in M1 and will be hardened in later milestones.
